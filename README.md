# VocabMaster

Android-приложение для изучения иностранных слов с флеш-карточками.
Русско-английский и англо-русский словарь: добавление слов, тренировка и
запоминание. Перевод новых слов — через Yandex Cloud Translate API.

## Стек

- Kotlin, Jetpack Compose
- Room (локальная база слов)
- Retrofit + OkHttp
- Hilt (DI)
- Yandex Cloud Translate API (получение IAM-токена по JWT сервисного аккаунта)

## Секреты (обязательно к прочтению)

Приложение обращается к Yandex Cloud от имени сервисного аккаунта. Эти
данные — **секретные** и в репозиторий не коммитятся. В исходниках
(`api/IamTokenProvider.kt`, `api/RetrofitInstance.kt`) оставлены только
плейсхолдеры:

- `SERVICE_ACCOUNT_ID`
- `KEY_ID`
- `PRIVATE_KEY`
- `FOLDER_ID`

Чтобы запустить приложение, получите свой авторизованный ключ в консоли
Yandex Cloud (сервисный аккаунт → создать авторизованный ключ) и подставьте
значения локально. Реальные секреты храните вне VCS (`local.properties`,
переменные окружения или защищённое хранилище) — `local.properties` уже
добавлен в `.gitignore`.
